# header

- 展示总卡片数量
- 展示当前展示流程卡片数量

# aside

- 双击、生成卡片

# nav

- 删除流程、增加流程

# canvas

- 展示当前流程

# branch

- 控制内容展开收起
- 编辑对应分支条件

# step

- 控制内容展开收起
- click 单选
- ctrl+click 进行多选
- 选中与当前展示不同
  - 选中卡片一定将卡片展示画布中
  - 而将卡片放置画布中不一定需要选中

# outline

- 展示当前流程大纲
- 点击卡片定位流程、定位卡片

# log

- 允许时展示运行步骤，以及输出对应 print
